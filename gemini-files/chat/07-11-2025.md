# 07 de novembro de 2025 - 08:07

## Pergunta:
O usuário reportou um erro ao executar `npm run dev` relacionado a classes de utilidade desconhecidas do Tailwind CSS (`border-border`, `bg-background`, `text-foreground`).

## Resposta:
Inicialmente, o problema parecia ser a sintaxe incorreta das diretivas `@tailwind` no `src/index.css`. Após a correção, o erro persistiu, indicando que o problema era de cache ou ordem de carregamento no pipeline de build do Vite/PostCSS, que impedia o Tailwind de reconhecer as classes customizadas baseadas em variáveis CSS. A solução proposta foi limpar todos os caches relacionados ao Vite e Tailwind CSS (`node_modules/.vite`, `.vite`, `dist`, `node_modules/.cache/tailwindcss`) e reiniciar o servidor de desenvolvimento.
